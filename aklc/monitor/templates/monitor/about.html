Monitoring system

This system monitors what is happening with Nodes (sensors etc.), repeaters & gateways that have been deployed. It provides the following principle functions:
  • Monitors items to check if they are online
  • Monitors battery information if available
  • Collects some basic data
  • Alerts people if nodes become unresponsive
  • Stores MQTT messages for later analysis/fault finding
  • Updates InfluxDB and Thingsboard with incoming data
